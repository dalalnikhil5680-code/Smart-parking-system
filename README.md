Smart Parking System

Overview:

An IoT-based Smart Parking System using ESP8266/Arduino and sensors to monitor parking slots.

Features:
	•	Detects vehicle parking and removal
	•	Sends Telegram notifications
	•	Displays available slots on OLED display
	•	Helps monitor parking status in real time

Components Used:
	1. ESP32 Microcontroller
2. IR Sensors (for slot detection)
3. Servo Motor (for barrier/gate control)
4. 16x2 LCD Display (with IC Module)
5. Buzzer
6. LEDs (Red and Green, for slot indicators)
7. 5V Power Supply / USB Power
8. Breadboard / PCB
9. Connecting Wires
10. Resistors
11. Jumper Wires
12. PVC Sheet / Enclosure

Outcome:

The system provides real-time parking information and alerts users when a vehicle is parked or removed.
